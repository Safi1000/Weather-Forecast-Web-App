WEATHER FORECAST WEB APP
kk
Features

1. Real-time weather data for any city
2. 7-day weather forecast
3. Displays key weather details: temperature, humidity, wind speed, etc.
4. User-friendly interface
5. Typing indicator for chatbot interaction
6. Error handling for invalid cities or API errors

Technologies Used
 
1. Frontend: HTML, CSS, JavaScript
2. API: OpenWeather API, Gemini API, Open Cage API


How to Run:

1. Open VS Code
2. Open project directory
3. Right click on "dashboard.html"
4. Click on "Open with Live Server"


               or

5. Open the project with the GITHUB Link: https://safi1000.github.io/Weather-Forecast-Web-App/
